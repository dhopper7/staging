Browser Support

jqDatepicker for PHP supports all major browsers:

  • Internet Explorer 6.x+
  • FireFox 3.x+
  • Safari 2.x+
  • Google Chrome 2.x+
  • Opera 9.5+

PHP

jqDatePicker is designed to work with PHP versions 5.1+.
