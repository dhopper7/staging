 jqGridRender Class
The jqGridRender Class is the most advanced and feature rich class jqGrid for PHP provides, aimed to do almost evething with the grid without writing any Javascript code. For users that want to write custom Javascript code we have developed a set of methods that can be used within PHP code.

The jqGridRender class extends the jqGridEdit Class (i.e. jqGrid Class too) . This mean that everthing that can be done with these classes is available in this class too.
