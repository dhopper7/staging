 Introduction

jqChart PHP is a server side component built on top of open and widely used standard - jQuery.

jqChart is a charting library written in PHP and JavaScript, offering an easy way of adding interactive charts to your web site or web application.

jqChart works in all modern browsers including the iPhone/iPad and Internet Explorer from version 6. Standard browsers use SVG for the graphics rendering. In Internet Explorer graphics are drawn using VML.

jqChart is based on native browser technologies and doesn't require client side plugins like Flash or Java.

jqChart supports line, spline, area, areaspline, column, bar, pie and scatter chart types. Any of these can be combined in one chart.

With the exporting module enabled, your users can export the chart to PNG, JPG, PDF or SVG format at the click of a button, or print the chart directly from the web page.
