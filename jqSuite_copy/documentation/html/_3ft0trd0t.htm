Browser Support

jqChart for PHP supports all major browsers:

  • Internet Explorer 6.x+
  • FireFox 3.x+
  • Safari 2.x+
  • Google Chrome 2.x+
  • Opera 9.5+

PHP

jqChart is designed to work with PHP versions 5.1+.

Databases (optional)

Currently jqChart for PHP supports the following databases:

  • MySQL 5.x+ (PDO)
  • PostgreSQL versions 7.x+ (PDO)
  • SQLite versions 3.x+ (PDO)
  • Microsoft SQL Server 2005+
  • Oracle 8+
  • MySQL (through Mysqli)
  • DB2 (IBM DB2 functions)
  • Array
  • MongoDB
