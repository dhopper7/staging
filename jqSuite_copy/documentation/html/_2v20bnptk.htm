 System requirements

Browser Support

jqGrid for PHP supports all major browsers:

  • Internet Explorer 6.x+
  • FireFox 3.x+
  • Safari 2.x+
  • Google Chrome 2.x+
  • Opera 9.5+

Note: a minor number of features may not work in all browsers. For example, currently RTL (right-to-left) support is limited to IE and FireFox only

PHP

jqGrid is designed to work with PHP versions 5.1+.

Databases

Currently jqGrid for PHP supports the following databases:

  • MySQL 5.x+ (PDO)
  • PostgreSQL versions 7.x+ (PDO)
  • SQLite versions 3.x+ (PDO)
  • Microsoft SQL Server 2005+ (sqlsrv)
  • Microsoft SQL Server 2005+ (PDO)
  • Oracle 8+
  • MySQL (through Mysqli)
  • DB2 (IBM DB2 functions)
  • ODBC (MSSQL, MySQL, PostgreSQL, Sqlite)
  • Array
  • MongoDB
  • Adodb (with suport of all drivers which are supported from adodb)

MySQL, PostgreSQL and SQLite databases are supported through PHP PDO (PHP Data Objects), so depending on your PHP setup you may need to install it before using jqGrid for PHP. PDO is by default installed in PHP 5.1+ distributions. PDO is a core PHP technology, created and supported by the core PHP team.
