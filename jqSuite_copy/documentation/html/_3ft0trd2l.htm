 Exporting
Trirand Inc offers the exporting web service free of charge. If you include the exporting module in your charts, two buttons will appear in the upper right. One button prints the chart, which is done on the client side only. The other button handles exporting. By default, an SVG representation of the chart is sent by POST to http://trirand.com/jqcharts/, where it is converted using Apache'sBatik converter to PDF, PNG or JPEG. SERVER
